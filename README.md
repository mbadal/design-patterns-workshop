# Design Patterns Workshop

## Project set up
Set up the project by running the commands:
```bash
cd ./docker
./ops up -d
```
After that, you can run PHP scripts with the command:
```bash

#Run PHP scripts
./ops php

# OR

#Run ANY bash command
./ops bash

# OR

#Log in directly to container
./ops container bash
```
